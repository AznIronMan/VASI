# VASI

Version: `0.10.1`
Last updated: `2026-07-12`

VASI is **Verified Authorized Signing Infrastructure**, a CNB product identity
maintained by Street Kings Productions for Clark & Burke LLC.

## Current Status

The repository was intentionally cleared on July 12, 2026. The imported
Documenso application, dependencies, deployment configuration, public
documentation, license files, and development tooling are no longer present in
the working tree.

Only these repository-root entries remain:

- `README.md`
- `AGENTS.md`
- `.private/` for untracked operator-only context
- `.git/` for repository history and metadata

This working tree does not contain a buildable, deployable, or
production-ready signing service. The prior VASI implementation and its pinned
Documenso `v2.14.0` baseline remain available through Git history.

## Private Material

The `.private/` directory is retained for local operator context. Its contents
must remain untracked and must not be copied into commits, logs, issues, or
public documentation.

## License

No application source or standalone license file is present in the cleared
working tree. Any future restoration or reuse of Documenso source must restore
and comply with the applicable upstream licenses, notices, and attribution.
